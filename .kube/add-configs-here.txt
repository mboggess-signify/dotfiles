# How to get kubectl configs
# 1. Have access to and log in to Rancher.
# 2. Choose a kube cluster.
# 3. In the top right corner, there's an icon that looks like a piece of paper
#    with its top right corner bent. Click that icon.
# 4. Config for this kube cluster should download.
# 5. Move the downloaded file to ~/.kube.
# 6. Repeat for each kube cluster.
# 7. Remove this file.
